# img-test Directory

This directory contains test images used for development and testing purposes.

## Getting Started

To work with this repository, follow these steps:

```bash
# Clone the repository
git clone https://github.com/Ilghar/email-assets.git

# Navigate to the repository
cd email-assets

# Checkout the main branch
git checkout main
```

## Purpose

The `img-test` directory is used for:
- Test images and uploads in progress
- Validating image hosting functionality
- Testing GitHub Pages deployment

## Usage

All images in this directory are publicly accessible via GitHub Pages at:
```
https://ilghar.github.io/email-assets/img-test/<filename>
```

For example:
- `https://ilghar.github.io/email-assets/img-test/email-96-test.png`
- `https://ilghar.github.io/email-assets/img-test/linkedin-96-test.png`

## Adding Test Images

1. Add your test images to this directory
2. Commit and push to the `main` branch
3. Wait 1-5 minutes for GitHub Pages to rebuild
4. Access your test images using the URL pattern above
