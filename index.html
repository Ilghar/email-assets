Email Assets

Use the folders below to reference hosted images from your HTML templates. Files are publicly available at https://ilghar.github.io/email-assets/<folder>/<filename>.

Available folders

  • icons/ – Icon assets
  • headers/ – Email header images
  • footers/ – Email footer images
  • signatures/ – Signature graphics
  • img-test/ – Test images and uploads in progress

Add new files by committing them to the main branch. GitHub Pages will refresh within a few minutes.
